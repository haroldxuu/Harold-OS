# Harold OS

Harold OS 是一套为第一代全球化中国职业人打造的人生操作系统。

它服务的人，是那些出生于普通家庭、没有海外资源、没有显赫背景，但希望依靠学习、职业能力、长期主义和全球化机会，建立更好事业、更好生活的人。

Harold OS 的第一位用户是 Harold 自己。这里的每一个框架、原则、清单和推荐，都是 Harold 经过真实生活验证后更新的。

## Mission

帮助第一代全球化中国职业人，在全球舞台建立事业、财富、健康、关系和人生复利系统。

## Philosophy

Harold OS is not written. It is lived.

## System Map

- 00 Vision：使命、定位、用户、长期愿景
- 01 Principles：跨领域长期原则
- 02 Health OS：健康、训练、睡眠、营养、恢复
- 03 Image OS：形象、衣橱、护肤、发型、全球消费决策
- 04 Career OS：全球化职业发展、Marketing、管理、汇报
- 05 AI OS：AI 工作流、第二大脑、自动化
- 06 Money OS：收入、储蓄、投资、全球资产配置
- 07 Relationship OS：亲密关系、家庭、社交
- 08 Life OS：旅行、阅读、人文、科学、使命
- 09 Decision OS：思维模型、决策框架、复盘
- 10 Gift OS：送礼、关系维护、全球礼物地图
- 99 Changelog：版本更新记录

## Development Workflow

Practice → System → Publish → Feedback → Improve

1. Harold 在真实生活中遇到问题。
2. 与 AI 互动，提炼这些问题背后的思考、原则、框架和操作手册。
3. Harold 作为主编进行 Review。
4. 内容更新到 GitHub。
5. 通过文章、视频和社区获得反馈。
6. 根据反馈继续迭代。
